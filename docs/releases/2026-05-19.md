**Environment:** Production (courtops.app)

## Staff Module — Phase 1 Complete (PR #32)

- **Shift Swap V1.** Staff can post shifts for take/swap, coworkers can claim, admin approves/denies.
- **Build-mode toggle.** Admin enters build mode to create/edit draft shifts. Non-build mode is read-only.
- **Edit draft shift time/notes.** Click a draft shift to modify times and notes inline.
- **TimeBlockPicker.** Visual drag-to-select time blocks for availability entry (replaces free-text typing).
- **Window share-link.** Availability windows get a shareable URL for staff.
- **Availability validation.** Prevents submissions outside club operating hours.
- **approximateHours fix.** Corrected hours calculation for availability with multiple time blocks.
